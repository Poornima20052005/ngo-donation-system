NGO Donation Platform is a full-stack web application developed using Django that enables users to contribute to social causes through online fundraising campaigns.

Features include user registration and authentication, campaign listing, donation tracking, and a dashboard displaying total donations and top-performing campaigns. The system manages donor information, campaign progress, and donation history using a relational database.

This project demonstrates backend development with Django, database operations, authentication systems, and dynamic web page rendering.

Technologies Used:
- Python
- Django
- HTML
